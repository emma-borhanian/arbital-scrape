The equivalence class of some $x \\in S$ is the set of elements of $S$ equivalent to $x$, written $[x]=\\{y \\in S \\mid x \\sim y\\}$, and we say that $x$ is a "representative" of $[x]$. We call the set of equivalence classes $S/\\sim = \\{[x] \\mid x \\in S\\}$.\n\nFrom the definition of an equivalence relation, it's easy to show that $x \\in [x]$ and $[x]=[y]$ if and only if $x \\sim y$.\n\nIf you have a set already [set_partition partitioned] into subsets ($S$ is the [-disjoint_union] of the elements of a collection $A$), you can go the other way and define a relation with two elements related whenever they're in the same subset ($x \\sim y$ when there is some $U \\in A$ with $x,y \\in U$).
