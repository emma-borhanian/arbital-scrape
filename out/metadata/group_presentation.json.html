However, the group corresponding to *this* presentation contains four elements, not two, so it is not $C_2$. This demonstrates the fact that if we have a presentation $\\langle X \\mid R \\rangle$, and a group can be written in such a way that all the relators hold in the group, and the group can be generated by the elements of $X$, that still doesn't mean the presentation describes the group; it could be that extra relations hold in the group that aren't listed in $R$. In fact, this group's elements are just words $x^n y^m$ for some integers $n, m$; this follows because, for instance, $xyx = xxy = x^2y$, and in general we can pull all the instances of the letter $x$ (and $x^{-1}$) out to the front of the word.
