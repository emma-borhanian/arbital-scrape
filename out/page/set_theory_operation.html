An operation in set theory is a Function of two sets, that returns a set.

Common set operations include [5s8 Union ($~$\cup$~$)], [5sb Intersection ($~$\cap$~$)], [5sc Relative complement ($~$\setminus$~$)], and [3xb Cartesian product ($~$\times$~$)].
