$~$\text{AIXI}^{tl}$~$ is a version of the ideal agent AIXI which only considers hypotheses of length $~$l$~$ that run for less than time $~$t$~$. A $~$tl$~$-bounded version of AIXI therefore only requires an unphysically large finite computer rather than an infinite hypercomputer.
