Given an element $~$g$~$ of a Group $~$G$~$, the conjugacy class of $~$g$~$ in $~$G$~$ is $~$\{ x g x^{-1} : x \in G \}$~$. It is the collection of elements to which $~$g$~$ is conjugate.
