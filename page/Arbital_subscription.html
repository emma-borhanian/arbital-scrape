Arbital allows you to subscribe to pages so you can receive updates when interesting changes occur. If you are logged in, you can subscribe by clicking on the "eye" icon. If the icon shows up blue, then you are already subscribed.

You will also automatically be subscribed to any thread you comment on.

Page subscriptions

You'll receive an update when:

  • The page is edited or its settings are changed.
  • The page has a new top level comment.
  • The page has a relationship change.

Comment subscriptions

You'll receive an update when:

  • The comment is replied to.
  • The comment is edited.
%%comment:### User subscriptions

You'll receive an update when the user:

- Creates a new page or writes a new comment.
- The user edits their page.%%
