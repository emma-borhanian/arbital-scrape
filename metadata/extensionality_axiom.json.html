A formal notation of the extensionality axiom can be written as:\n\n$$ \\forall A \\forall B : ( \\forall x : (x \\in A \\iff x \\in B) \\Rightarrow A=B)$$\n\n##Examples\n\n - $\\{1,2\\} = \\{2,1\\}$, because whatever object we choose, it either belongs to both of these sets ($1$ or $2$), or to neither of them (e.g. $5$, $73$)\n\n%%comment:\n \n- If $A = \\{x \\mid x = 2n \\text{ for some integer } n \\}$ and $B = \\{x \\mid x \\text{ is even } \\}$, then $A=B$.
